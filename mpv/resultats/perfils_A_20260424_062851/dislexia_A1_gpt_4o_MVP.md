La fotosíntesi és un procés de les plantes. Les plantes fan menjar amb llum. Les plantes necessiten llum, aigua i aire. La fotosíntesi passa a les fulles verdes.

Les plantes agafen aire per les fulles. Les plantes agafen aigua per les arrels. La llum del sol ajuda a fer el menjar.

La reacció és: aire + aigua + sol → menjar + oxigen

Les plantes fan menjar amb aire i aigua. Les plantes alliberen oxigen. Les persones necessiten oxigen per viure. Les plantes fan molt oxigen per a la Terra.
