01 El docent configura

TEXT ORIGINAL

▼
Enganxa o puja el text a adaptar. S'envia com a user message a Gemini i tambe com a query RAG.
El text que el docent vol adaptar. S'envia com a missatge d'usuari a Gemini (separat del prompt/system).

Tambe s'usa per construir la query de cerca al corpus RAG: les primeres 200 paraules + les caracteristiques actives formen la consulta de cerca vectorial.

CONTEXT DOCENT

▼
Etapa, Curs, Ambit, Materia, Tipus aula, Genere discursiu
On va al prompt? A la seccio CONTEXT — Gemini ho sap pero no son ordres.

Excepcio: el genere discursiu (explicatiu, narratiu, instructiu, argumentatiu) activa instruccions especifiques del corpus.

L'etapa determina el MECR per defecte (Infantil=A1, Primaria=B1, ESO/Batx=B2).

02 El sistema calcula
AUTOMATIC (el docent pot modificar)

PROPOSTA

▼
DUA: Acces / Core / Enriquiment · MECR sortida: pre-A1...B2 · Complements
El sistema calcula automaticament a partir del perfil:

• DUA — DI sever o TEA nivell 3 o nouvingut pre-A1 → Acces. Altes cap. → Enriquiment. Resta → Core.
• MECR sortida — PARAMETRE MES IMPORTANT: controla TOTES les instruccions NIVELL (max paraules/frase, subordinades, temps verbals, to...)
• Complements — glossari, esquema, preguntes... segons perfil

FILTRE + MACRODIRECTIVES

▼
1. Selecciona per perfil + MECR · 2. Aplica sub-variables · 3. Suprimeix redundants · 4. Intensifica · 5. Agrupa en macrodirectives
4 criteris d'activacio:

SEMPRE (12) — Bones practiques universals
NIVELL (31) — Depenen del MECR sortida
PERFIL (38) — Per caracteristiques actives + sub-variables
COMPLEMENT (3) — Pictogrames, esquema, mapa

Supressio intel·ligent: si dislexia activa, A-01/A-03 no s'envien (H-08 ja ho cobreix). Si TEA actiu, A-05 no s'envia (H-02 ho cobreix).

Intensificacio: tdah.baixa_memoria_treball → chunking 3 elements (no 5). tdah.grau=sever → micro-blocs 2-3 frases (no 3-5).

PERSONA-AUDIENCE

▼
"Alumne de 14 anys, nouvingut, L1 arab, CALP inicial, alfabet no llati..."
Narrativa natural generada a partir de les sub-variables NARRATIVA.

Exemple real:
Escrius per a un alumne de ESO (2n).
Alumne nouvingut que parla arab com a L1.
Alfabet no llati: necessita transliteracio fonetica.
Escolaritzacio previa parcial o interrompuda.
Llengua academica (CALP) inicial: no domina el registre escolar ni vocabulari abstracte.
Nivell MECR de sortida: A1.
No son ordres — son context perque l'LLM entengui per a qui escriu.

03 Prompt assemblat (el que veu l'LLM)

CAPA 1: IDENTITAT

fixa — "Ets ATNE de Jesuites Educacio"▼
Sempre igual:
"Ets ATNE de Jesuites Educacio. Restriccions: NO inventis contingut. NO meta-text. Preserva termes tecnics. Adapta, no crees."
CAPA 2: MACRODIRECTIVESvariable — 6-8 blocs
LEXIC SINTAXI ESTRUCTURA COGNITIU QUALITAT PERSONALITZACIO PERFIL
▼
Exemple real (TDAH moderat B1):
LEXIC: Usa vocabulari frequent. Termes tecnics en negreta. Un terme = un concepte. Referents pronominals explicits. Elimina idiomatiques...

SINTAXI: Una idea per frase. Veu activa. Subjecte explicit. Max 12-18 paraules...

ESTRUCTURA: Paragrafs curts. Titols descriptius. Indicadors progres [X de Y]...

PERFIL: TDAH: micro-blocs 3-5 frases. Retroalimentacio visual. Variacio dins text.
L'LLM no veu IDs (A-01, H-04...) — nomes prosa tematica agrupada.

CAPA 3: CONTEXT

Educatiu + Persona + RAG + Complements▼
Aqui NO hi ha ordres — son informacio de context:

3a. Context educatiu: Etapa: ESO | Curs: 3r | Materia: Biologia
3b. Persona-audience: Alumne nouvingut arabofon, CALP inicial, TEA nivell 2
3c. Coneixement pedagogic (RAG): 8-12 fragments del corpus FJE
3d. Complements activats: glossari, esquema, preguntes...

CAPA 4: FORMAT SORTIDA

## Text adaptat + seccions opcionals▼
Sempre: ## Text adaptat + ## Argumentacio pedagogica
Si actius: ## Glossari (taula bilingue) · ## Esquema visual · ## Mapa conceptual · ## Preguntes comprensio (3 nivells) · ## Bastides · ## Activitats aprofundiment

Cada complement te instruccions detallades de format.

Com funcionen les macrodirectives
  1. El cataleg te 88 instruccions individuals (A-01, A-02, H-04...) amb regles d'activacio.
  2. El filtre selecciona nomes les rellevants pel cas (perfil + MECR + sub-variables).
  3. Les agrupa en 6-8 blocs tematics (LEXIC, SINTAXI, ESTRUCTURA...) sense IDs.
  4. L'LLM rep 6-8 blocs coherents en lloc de 45+ regles soltes → millor seguiment.
