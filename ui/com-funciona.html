Com funciona ATNE

De text curricular a material adaptat, en 3 passos

ATNE és un assistent que adapta textos educatius al perfil de cada alumne. Entendre com funciona ajuda a treure-li el màxim partit.

01

El flux de treball

ATNE s'organitza en tres passos seqüencials. Cada pas té una funció clara i es pot repetir tantes vegades com calgui.

1

Per a qui

Selecciones o crees el perfil de l'alumne o grup. El perfil defineix el nivell lingüístic, les condicions i els ajuts que s'aplicaran.

2

El text

Enganxes, puges o generes el text que vols adaptar. Configures la matèria, el nivell educatiu i els complements opcionals.

3

Adaptacions

Reps el text adaptat amb tots els ajuts activats. Pots refinar-lo, corregir-lo, comparar-lo amb l'original i exportar-lo.

Els perfils es desen al teu compte i es recuperen des de qualsevol dispositiu. Un cop creat el perfil d'un alumne, el reutilitzes en totes les adaptacions futures.

02

Els perfils: qui és l'alumne

El perfil és la peça clau d'ATNE. Com més precís sigui, millor serà l'adaptació. Es construeix a partir del que observes a l'aula, no necessàriament d'un diagnòstic formal.

  • ✓Nivell MECR: Del A1 (inicial) al C1 (competent). Defineix la complexitat lingüística de l'adaptació.
  • ✓Conductes observades: Es distreu amb textos llargs, fatiga lectora, dificultats amb connectors… L'app activa els ajuts adequats automàticament.
  • ✓Condicions i diagnòstics: Opcional. Si tens informe o PI, pots afegir TDAH, dislèxia, L2, AACC, TEA, etc.
  • ✓Sub-variables: Per a cada condició apareixen camps addicionals: per a L2, la llengua familiar i el temps a Catalunya; per a AACC, el tipus de superdotació.

Les condicions disponibles:

TDAH Dislèxia / DEA Català com a L2 Altes capacitats TEA Discapacitat intel·lectual Discapacitat auditiva Discapacitat visual

També pots crear perfils de grup per adaptar un text a tota la classe amb múltiples nivells simultàniament (Bàsic · Estàndard · Avançat).

03

Com treballa la intel·ligència artificial

ATNE usa un model de llenguatge gran (LLM) —Gemma 4 31B— guiat per un catàleg de més de 90 instruccions pedagògiques específiques per a cada condició i nivell. No és un xat lliure: cada instrucció ha estat validada amb el marc FJE.

👤
Filtratge d'instruccions
A partir del perfil i el MECR, el sistema selecciona automàticament les instruccions rellevants del catàleg (unes 40 de 90+).
↓
🤖
Generació (LLM)
L'LLM adapta el text seguint les instruccions seleccionades. Típicament tarda 15-30 segons.
↓
🔍
Verificació automàtica
Un segon pas de l'LLM avalua si l'adaptació compleix els criteris. Si la puntuació és baixa, es regenera.
↓
📝
Pipeline de qualitat
LanguageTool corregeix ortografia i gramàtica catalana. Es detecten paraules sospitoses i caràcters exòtics.

🔒 Privacitat: Els textos s'envien a l'API de l'LLM per processar-los però no s'emmagatzemen permanentment. ATNE no desa el contingut dels textos originals ni adaptats, només metadades anonimitzades per a millora del sistema.

04

Els complements

A més del text adaptat, pots activar complements opcionals que l'LLM genera simultàniament:

📖

Glossari

Termes clau del text amb definicions accessibles per a l'alumne.

❓

Preguntes de comprensió

Preguntes graduades en 3 nivells (bàsic, estàndard, avançat) sobre el text adaptat.

🗂️

Esquema visual

Resum jeràrquic del contingut. Útil per a alumnat que necessita veure l'estructura abans de llegir.

🪜

Bastides lingüístiques

Suports per a la comprensió i producció textual, seguint el model MALL/TILC de FJE.

05

Eines de revisió i exportació

Un cop tens l'adaptació, el Pas 3 ofereix diverses eines per revisar i distribuir el material:

  • AaTipografia: Canvia la font (Atkinson, OpenDyslexic, Arial…) per a alumnat amb dislèxia o necessitats visuals.
  • ⟺Vista comparativa: Original i adaptat costat a costat. Útil per verificar que no s'ha perdut contingut curricular.
  • ✏️Refinar: Escriu instruccions concretes i l'IA reescriu el text incorporant les teves indicacions.
  • 📋Rúbrica pedagògica: Avalua l'adaptació i demana a l'IA que millori els aspectes febles.
  • ⬇Exportació: PDF (llest per imprimir), Word (.docx, editable) o text pla.

Ara que saps com funciona, prova-ho amb el teu primer text.
