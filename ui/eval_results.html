Qui te rao?

Quan el generador s'autoavalua (Gemini), dona HC 66%. Quan dos jutges externs avaluen (Claude), donen RAG 60-70%. La discordancia (40-80% de desacord amb Gemini) es un indicador classic de biaix d'autoavaluacio documentat a la literatura (Liu et al., 2023; ACL 2025).

Recomanacio: MAI usar el generador com a jutge. Sempre jutge extern.

Gemini — Per criteri (200 casos)

C2 Adequacio al perfil es on RAG s'acosta mes (HC 107 vs RAG 74). Les instruccions detallades per perfil produeixen diferencies observables, pero Gemini segueix preferint HC globalment per biaix.

C3 Preservacio curricular te 53 empats — ambdues branques preserven be el contingut original.

Bugs detectats per l'avaluacio

BUG 1: Gemini ignora instruccions de glossari bilingue (G-01, G-03, E-04)
Cap de les dues branques genera traduccions a l'arab, xines o wolof al glossari. Les instruccions estan al prompt RAG pero Gemini les ignora.
FIX: Reforcar amb few-shot example bilingue explicit al prompt. Afegir verificacio post-processament.
BUG 2: Gemini ignora supressions per Altes Capacitats (A-01, A-03, A-05...)
Per P7 (AC/Enriquiment), ambdues branques simplifiquen (parentesis explicatius) quan haurien d'enriquir. Les 7 regles suprimides al filtratge no tenen efecte.
FIX: Fer supressions explicites amb instruccions negatives en MAJUSCULES: "NO simplifiquis. NO afegeixis definicions parentetiques."
BUG 3: Biaix d'autoavaluacio de Gemini confirmat
Gemini prefereix HC en 9/10 perfils. Jutges externs (Claude) prefereixen RAG en 5-7/10. Concordancia Gemini-Opus: 20%.
FIX: MAI usar el generador com a avaluador. Jutge extern obligatori (Claude o huma).
