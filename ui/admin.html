Selector de model per fase

Cada fase del pipeline pot fer servir un model diferent. Els canvis s'apliquen en calent a totes les peticions noves (les en curs acaben amb el model anterior).

Fix: una sola crida sempre amb el mateix model. Rotació silenciosa: cada crida tria aleatòriament un dels models seleccionats. Serveix per A/B cec al pilot i recollir estadístiques de quin model funciona millor. El docent no veu quin model s'ha usat.
